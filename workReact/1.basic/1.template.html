vscode에서 자주 사용되는 단축키

  1. html문서 템플릿 작성법->html 파일 작성후 ! enter or html:5 enter
  2. 태그 작성하는 방법=>태그이름 enter => 자동완성기능 ex) div enter
  3. 태그를 여러개 생성하는 방법->태그명*n(반복할 수) ex) div*5 =>div+div+div+div+div
  4. 태그 중첩해서 생성하는 방법=>태그명>태그명2>태그명3 ex) div>ul>ol*3
        1. 인접한 태그 생성하는 방법=>태그명+태그명2+태그명3 ex) div+h1+h2+h3+span

        2. 아이디 선택자 설정=>태그명#아이디명 ex) div#name 생성=>뒤에 div#name{내용추가}
          내용추가
        3. 클래스 선택자설정=>태그명.클래스명 ex) div.test
        4. 태그{입력하고 싶은 텍스트} 입력후 enter or tab ex) p{안녕하세요}

          안녕하세요

        5. 숫자삽입(숫자 1씩증가) =>태그명#아이디명$*n, 태그명.클래스명$*n, 태그명{텍스트$}.클래스명$*n ex) div{텍스트$}.test$*3
          텍스트1
          텍스트2
          텍스트3
        6. ^p 앞쪽 1개 태그 건너 계층과 동일계층에 배치(여러번 사용가능)=>^태그명 ex) div>ul>li^p

        7. ^^p 앞쪽 2개 태그 건너 계층과 동일계층에 배치=>^^태그명 ex) div>ul>li^^p

        8. +p 동일계층에 추가=>+태그명 ex) div>ul>li+p

        9. (태그>태그,,,) 그룹단위 배치, 맨 마지막 그룹은 괄호 생략 필수 ex) div>(ul>li)+ol>li
        10. 자동 완성 div#myid.classA.classB 입력후 enter (id와 클래스를 동시에 생성) ex) div#myid.classA.classB
